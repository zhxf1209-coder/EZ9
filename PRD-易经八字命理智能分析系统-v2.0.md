# 易经命理产品PRD文档

## 产品概述

**产品名称**：易经八字命理智能分析系统 v2.0

**文档版本**：V1.0

**文档日期**：2026年03月20日

---

## 一、产品愿景与目标

### 1.1 产品愿景

打造一款融合传统文化智慧与现代科技体验的命理服务产品，通过AI智能分析技术，将渊海子平、三命通会等古典命理典籍的理论精髓转化为可视化、可操作的人生指导工具，让每一位用户都能获得专业、精准、富有东方美学韵味的命理服务体验。

### 1.2 产品目标

#### 核心目标

- 建立命理服务领域的专业标杆，以精准的分析和优雅的界面设计脱颖而出
- 通过多元化的功能矩阵，提升用户粘性与回访率，打造命理服务闭环
- 秉承"命理普惠"理念，所有核心功能免费开放，让传统文化智慧惠及大众

#### 商业目标

- 积累高质量用户群体，建立用户信任与品牌认知
- 形成稳定的用户活跃度，日活用户达到注册用户的30%以上
- 为后续增值服务奠定用户基础

### 1.3 产品定位

融合"古典韵味"与"科技质感"的智能命理分析平台，强调专业性、权威性与美观性的统一。

## 二、功能详细设计

### 2.1 婚姻合婚评测功能

#### 2.1.1 输入参数

双方档案信息（共用命理测算档案格式）：

- 姓名（用于报告显示）
- 性别
- 出生年月日时（精确到时辰）
- 出生地（用于真太阳时校正）

#### 2.1.2 合婚分析维度

**一、命格基础对比**

1. 四柱干支对照表
2. 五行配置分析
3. 日主强弱判定

**二、性格契合度分析**

1. 性格特质对比
2. 沟通模式评估
3. 价值观契合度

**三、婚姻运势走势**

1. 相识相恋期
2. 新婚磨合期
3. 稳定发展期
4. 中年转折期
5. 白头偕老期

**四、相处建议**

1. 和谐相处指南
2. 性格差异调和
3. 日常互动优化

**五、注意事项与风险提示**

1. 需特别留意的事项
2. 岁运影响分析
3. 健康与财运关联

#### 2.1.3 合婚指数算法

```
总分100分，≥80分为上等婚，60-79分为中等婚，<60分为下等婚

权重分配：
- 日主关系：30%
- 五行互补：25%
- 四柱合冲：20%
- 婚姻宫配置：15%
- 子息缘分：10%
```

### 2.2 今日运势功能

#### 2.2.1 核心计算维度

| 维度 | 计算因子 | 权重 |
|-----|---------|-----|
| 综合运势 | 日主旺衰+日干生克+吉神凶煞 | 100% |
| 事业运势 | 官星配置+印星扶身 | 30% |
| 感情运势 | 财星旺度+桃花星 | 25% |
| 财运运势 | 财星得令+食伤生财 | 25% |
| 健康运势 | 五行平衡+凶煞侵扰 | 20% |

#### 2.2.2 用户粘性机制

**每日签到功能**：

- 每日首次进入自动签到
- 连续签到累积运势能量值
- 能量值可兑换高级功能

**签到奖励体系**：

| 连续签到天数 | 奖励内容 |
|------------|---------|
| 1天 | 运势能量+10 |
| 3天 | 运势能量+30 |
| 7天 | 运势能量+80+小吉签文 |
| 30天 | 运势能量+500+大吉签文+专属开运壁纸 |

### 2.3 年度运势功能

#### 2.3.1 年度运势报告内容

**一、年度总纲**

- 流年概述
- 年度主题词
- 年度运势曲线

**二、各维度详细分析**

- 事业运势
- 财运运势
- 感情运势
- 健康运势

**三、实用开运信息**

| 信息类型 | 内容说明 |
|---------|---------|
| 吉祥方位 | 年度最利方位 |
| 幸运颜色 | 年度幸运色 |
| 幸运数字 | 每日幸运数字 |
| 开运植物 | 利于本年的植物 |
| 贵人属相 | 年度贵人星座/属相 |
| 吉祥数字日 | 每月最适宜做大事的日期 |

### 2.4 财运分析功能

#### 2.4.1 财运分析内容

**一、财富命格基础分析**

- 财星配置诊断
- 财富层次判定
- 财运特点画像

**二、正财偏财详细分析**

- 正财分析
- 偏财分析

**三、投资理财建议**

- 投资风格适配
- 投资领域推荐
- 理财时机提示

**四、破财风险预警**

- 破财风险识别
- 高风险时间段
- 破财化解方案

### 2.5 化解建议功能

#### 2.5.1 化解体系设计

**一、命局诊断与问题识别**

- 五行缺失调理
- 五行过旺调节
- 冲克刑害化解

**二、化解方案分类**

1. 风水调理建议
2. 饰品佩戴指南
3. 颜色调整方案
4. 数字选择指南

## 四、技术需求

### 4.1 核心算法

**八字排盘算法**：

1. 日期转换：阴历→阳历
2. 真太阳时校正
3. 排四柱（年、月、日、时）
4. 十神计算
5. 神煞判断
6. 五行统计
7. 旺衰判定

**合婚匹配算法**：

1. 日主关系分析
2. 五行互补评估
3. 四柱合冲判定
4. 婚姻宫配置分析
5. 子息缘分评估

### 4.2 性能要求

| 指标 | 要求 |
|-----|-----|
| 八字排盘响应时间 | ≤200ms |
| 简单报告生成时间 | ≤1s |
| 复杂报告生成时间 | ≤3s |
| 系统并发处理能力 | ≥1000 QPS |
